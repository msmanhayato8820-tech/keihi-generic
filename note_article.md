# 楽楽精算の1/100のコストで動く経費精算システムを作ったので公開します

**GAS×Next.js製・承認フロー・OCR・Teams/Slack通知・7種ストレージ対応**

---

## なぜ作ったか

経費精算って、中小企業にとって地味にしんどい業務ですよね。

紙の領収書を集めて、Excelに手入力して、上長にハンコもらって、経理に回して...。ミスも多いし、月末に地獄を見る。かといって「楽楽精算」や「マネーフォワード経費」を導入しようとすると、月額3万円〜。年間36万円以上のランニングコストがかかります。

社員5人の会社に、年間36万の経費精算SaaSは正直キツい。

でも「Excel運用のまま」も限界がある。この**空白地帯**を埋めるために、自分で作りました。

結論から言うと、**Vercel（無料）+ Google Apps Script（無料）で月額0円**で動く経費精算システムができました。楽楽精算の1/100どころか、0円です。

## 機能一覧

「無料で動く」と聞くと、しょぼいシステムを想像されるかもしれません。でも、中小企業の経費精算に必要な機能はひと通り揃えました。

### 承認ワークフロー
- **4段階ロール制御**: 一般社員 → 部長承認 → 経理承認 → 完了
- 金額やカテゴリに応じた承認ルート自動分岐（5万円超は役員承認など）
- 承認・差戻しの履歴はすべてタイムスタンプ付きで記録
- 電子帳簿保存法を意識したタイムスタンプ自動付与

### 領収書OCR
- **Tesseract.js**: ブラウザ内で完結。無料・オフライン対応
- **Gemini API**: Google AIによる高精度読取。手書き領収書もOK
- 設定画面からワンクリックで切り替え可能

### 通知機能（3チャネル）
- **Slack**: Incoming Webhook で申請・承認時に自動通知
- **Microsoft Teams**: Adaptive Card形式で通知（Slack同等の機能）
- **メール**: GAS経由のリレー送信対応
- すべて設定画面のGUIから設定可能。コード編集不要

### クラウドストレージ（7種対応）
領収書の自動アップロード先を選べます。
- AWS S3
- Google Drive
- Microsoft OneDrive
- Box
- Dropbox
- Backblaze B2（S3互換・格安）
- Cloudflare R2（S3互換・エグレス無料）

すべてGUI設定。APIキーを入力するだけで使えます。

### 会計ソフト連携
- **マネーフォワード / freee / 弥生** に対応した仕訳CSV出力
- 勘定科目・税区分のマッピング済み
- 適格請求書（インボイス）番号の管理にも対応

### GUI管理画面
- 会社名・ロゴ・テーマカラーを設定画面から変更
- デモアカウントの追加・削除・編集
- 経費カテゴリの追加・編集・並び替え
- 税率の追加・編集（インボイス制度や税制改正に対応）
- 取引先マスタ管理
