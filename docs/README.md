## 最短セットアップ（5分）

とりあえず使い始めたい場合は、以下の2ステップだけでOKです。

1. **[Vercelへのデプロイ](01_vercel_deploy.md)**（5分）
2. **[カスタマイズ](05_customization.md)**（3分） - 会社名を変更

この時点でデモモードとして動作します。データの永続保存が必要になったら、[GASの設定](02_gas_setup.md) に進んでください。

## フルセットアップ（30分）

すべての機能を有効にする場合は、上から順番に1〜5を実施してください。合計約30分で完了します。
