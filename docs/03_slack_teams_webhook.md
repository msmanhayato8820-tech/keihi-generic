# Slack・Teams通知の設定手順

| 所要時間 | 難易度 |
|---------|--------|
| 約5分（各サービス） | ★★☆（ふつう） |

経費の申請・承認・差戻し時に、SlackやMicrosoft Teamsに自動で通知を送る設定です。両方設定することもできますし、片方だけでもOKです。

---

## Slack通知の設定

### 1. Slack Appを作成する

1. https://api.slack.com/apps にアクセスします
2. 「Create New App」をクリックします
3. 「From scratch」を選択します
4. App名を「経費精算通知」、ワークスペースを自社のSlackワークスペースに設定します
5. 「Create App」をクリックします

![手順1](images/slack_create_app.png)

### 2. Incoming Webhookを有効にする

1. 左メニューから「Incoming Webhooks」をクリックします
2. 「Activate Incoming Webhooks」をオンにします
3. ページ下部の「Add New Webhook to Workspace」をクリックします
4. 通知を送信するチャンネル（例: #経費精算）を選択します
5. 「許可する」をクリックします
6. **Webhook URL**が生成されます。このURLをコピーしてください

![手順2](images/slack_webhook.png)

> Webhook URLは `https://hooks.slack.com/services/T.../B.../...` のような形式です。

### 3. システムに設定する

1. 経費精算システムにログインします（管理者アカウント）
2. 「設定」→「Slack通知設定」セクションを開きます
3. コピーしたWebhook URLを貼り付けます
4. 「Slack設定を保存」をクリックします
5. 「テスト送信」をクリックして、Slackにメッセージが届くことを確認します

![手順3](images/slack_setting.png)

## Microsoft Teams通知の設定

### 1. Teamsチャネルにコネクタを追加する

1. Microsoft Teamsを開きます
2. 通知を受け取りたいチャネル（例: 「経費精算」チャネル）を右クリックします
3. 「コネクタ」または「ワークフロー」を選択します
4. 「Incoming Webhook」を検索して追加します

> **注意**: Teamsのバージョンによって手順が異なります。新しいTeamsでは「ワークフロー」からWebhookを作成する場合があります。

![手順1](images/teams_connector.png)

### 2. Webhook URLを取得する

**従来のTeams（コネクタ方式）の場合:**
1. 「Incoming Webhook」の「構成」をクリックします
2. 名前を「経費精算通知」に設定します
3. 「作成」をクリックします
4. 表示されたURLをコピーします

**新しいTeams（ワークフロー方式）の場合:**
1. 「チャネルに投稿する場合」のテンプレートを選択します
2. ワークフロー名を「経費精算通知」に設定します
3. 投稿先のチャネルを選択します
4. 「作成」をクリックします
5. 表示されたWebhook URLをコピーします

![手順2](images/teams_webhook.png)

### 3. システムに設定する

1. 経費精算システムにログインします（管理者アカウント）
2. 「設定」→「Microsoft Teams通知設定」セクションを開きます
3. コピーしたWebhook URLを貼り付けます
4. 通知タイミングのチェックボックスを設定します
   - **申請時**: 経費が申請されたときに通知
   - **承認時**: 経費が承認されたときに通知
   - **差戻し時**: 経費が差し戻されたときに通知
5. 「Teams設定を保存」をクリックします
6. 「テスト送信」をクリックして、Teamsにメッセージが届くことを確認します

![手順3](images/teams_setting.png)

## 通知のタイミング

通知は以下のタイミングで自動送信されます。

| タイミング | Slack | Teams | 通知内容 |
|-----------|-------|-------|---------|
| 経費申請時 | 常時 | チェックON時 | 「新しい経費申請: ○○さんが「△△」¥X,XXXを申請しました」 |
| 経費承認時 | 常時 | チェックON時 | 「経費承認完了: 「△△」¥X,XXXが承認されました」 |
| 経費差戻し時 | 常時 | チェックON時 | 「経費差戻し: 「△△」¥X,XXXが差し戻されました」 |

---

## うまくいかない場合

| 症状 | 対処法 |
|------|--------|
| テスト送信で通知が届かない | Webhook URLが正しいか確認してください |
| Slackの通知が届かない | Slackアプリがチャンネルに参加しているか確認してください |
| Teamsの通知が届かない | コネクタが有効になっているか確認してください。Teams管理者がコネクタを無効にしている場合があります |
